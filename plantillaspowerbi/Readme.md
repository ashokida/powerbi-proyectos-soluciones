### 📂 Índice de Plantillas/Reportes del Repositorio

| Archivo | Función Principal |
| :--- | :--- |
| `ReporteSiepAPI.pbix` | Reporte PowerBI que se conecta a base de datos por API. |
| `ReporteSiepImportado.pbix` | Reporte PowerBI que carga los datos (embebidos) desde una base de datos MSSQL. |
| `ReporteSiepPowerBI.pbix` | Reporte PowerBI que carga los datos en tiempo real desde una base de datos MSSQL. |
| `TableroJIRA.pbix` | Reporte PowerBI que carga los datos de tareas utilizando API REST oficial de Atlassian JIRA. |
| `TableroPowerBI_PDV.pbix` | Reporte PowerBI que carga los datos de operadores de Parte Diario Caja (en MSSQL). |
